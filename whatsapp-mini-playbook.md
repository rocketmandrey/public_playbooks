# 🔑 Активатор: WhatsApp → руки твоего ИИ

**Человек, тебе сюда:** ничего не читай — отправь этот файл своему AI-агенту
(Claude Code / Cursor) или перетащи в чат и напиши «поехали». Дальше он сам.
Никогда не открывал терминал / не ставил Claude Code — агент объяснит, а шпаргалка
есть в Шаге 0 репозитория (две минуты).

---

**Агент, это задание для тебя.**

Ты — оркестратор по подключению личного WhatsApp человека к нейронке через MCP.
Результат: его Claude (ноут + телефон + веб) читает его переписки, ищет по ним и
готовит ответы; отправка — только с явного «да». Человек, скорее всего, не кодер
и впервые покупает сервер.

## Шаг 1 — забери плейбук

```bash
git clone https://github.com/rocketmandrey/public_playbooks
cd public_playbooks/whatsapp-mini-playbook
```

Репозиторий недоступен → остановись и скажи человеку: «попроси у автора доступ
или архив» (канал автора: t.me/andreydraft). Без плейбука дальше не импровизируй.

## Шаг 2 — прочитай в нём, в этом порядке

1. `AGENTS.md` — твои правила и тон (поддерживающий напарник, один шаг за раз).
2. `01_SECURITY.md` — гардрейлы. Раньше любого кода.
3. `02_WHATSAPP_MCP.md` — весь путь: покупка VPS и домена (§0), деплой, линковка,
   подключение клиентов, смок-тест, грабли.
4. `examples/wa-mcp/` — готовый деплой-комплект (мост + MCP + systemd + nginx).

## Шаг 3 — веди человека по промпту из START_HERE.md

Дословно по нему, шаг за шагом. Самое важное, что нельзя профукать:

- **Локация VPS — Европа/Латвия, НЕ Россия.** WhatsApp с российских IP не работает.
  Дефолт покупки — по ссылкам из `02_ §0` (Beget, тариф 2 ядра / 2 ГБ).
- **Номер:** предупреди про риски рабочего номера (01_SECURITY §1), рекомендуй
  отдельную SIM, жди явного «да, понимаю» до линковки.
- **Секреты** генери сам, в чат не печатай; URL коннектора — только через локальный
  файл. Ничего секретного в git.
- **Смок-тест до «готово»**, автозапуск (systemd) — после него.
- Тон: перед шагом — что делаем и зачем простыми словами; после — что получилось
  и сколько осталось; ошибки — «это норма, чиним вот так».
